Match Making Simulator 2

This simulator tries to add continuous equalization matchmaking. When a new policy is added, it immediately plays a bunch of games involving that policy. When a policy is removed, it removes all past games that included that policy. The goal is to equalize how many games are played between the active policies.

Versus Win Percentage

Each game splits its eight players into a crew team of six and an imposter team of two. Each side blends its lowest player's actual rank with its team average rank, and the higher score wins. The slider below sets the blend: at 0 only the weakest link matters, at 1 only the team average matters, so higher values let strong policies carry the game in spite of a weak teammate. The random factor adds bell-curve (Gaussian) noise to each side's score before picking the winner; the chosen ± rank covers about 95% of the draws. Each cell shows how often the row policy's side beat the column policy's side when they played on opposite teams.

Sort Order

How close the simulated ordering has gotten to the ground truth. Active policies are sorted by observed win percentage; the ground truth rank column shows each policy's true rank, so a perfect ordering reads top to bottom in descending rank.
